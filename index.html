What is Project Wyvern?

Project Wyvern is a unified coordination network designed to connect individuals across seven structured factions. It serves as a hub for political discussion, internal news, and classified intelligence distribution.

Access is tiered by faction and clearance level. All activity within the network is logged and monitored by Command.
